# StingRay

[Ray Tracing in One Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html) written in Zig 0.16.0.

Linear algebra via [zlm](https://github.com/HTRMC/zlm).

## Build

```sh
zig build
```

## Run

```sh
zig build run > image.ppm
```
